# 榫卯非遗文化传承平台 — 完整测试检验流程

## 一、测试环境准备

### 1.1 环境要求

| 组件 | 版本要求 | 说明 |
|------|---------|------|
| JDK | 1.8+ | Java 运行环境 |
| MySQL | 5.7+ | 数据库服务 |
| IntelliJ IDEA | 任意版本 | 后端开发/调试 |
| DBeaver | 任意版本 | 数据库管理工具 |
| 微信开发者工具 | 稳定版 | 小程序前端调试 |
| Maven | 3.6+ | 依赖管理（IDEA 内置） |

### 1.2 数据库准备（使用 DBeaver）

1. **打开 DBeaver**，连接到本地 MySQL 数据库
2. **创建数据库**：
   ```sql
   CREATE DATABASE ljx_platform CHARACTER SET utf8 COLLATE utf8_unicode_ci;
   ```
3. **执行初始化脚本**：
   - 在 DBeaver 中右键点击 `ljx_platform` 数据库 → `SQL编辑器` → `打开SQL控制台`
   - 打开 `ljx_backend/sql/schema_mysql5.sql` 文件
   - 点击执行按钮（或按 `Ctrl+Enter`）
   - **注意**：如果 MySQL 版本低于 5.6.4，需将 `FULLTEXT KEY` 改为普通 `KEY`，否则会出现 `The used table type doesn't support FULLTEXT indexes` 错误

4. **确认数据库连接配置**（`application.yml`）：
   ```yaml
   url: jdbc:mysql://localhost:3306/ljx_platform?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
   username: root
   password: root
   ```

## 四、前后端联调测试

### 4.1 联调前提

- 后端服务在 IDEA 中正常运行（端口 8081）
- 微信开发者工具已开启"不校验合法域名"
- 小程序代码中的请求地址已改为 `http://localhost:8081/api/xxx`（或你电脑的局域网 IP）

### 4.2 联调测试项

目前小程序前端代码中硬编码了本地数据，尚未调用后端 API。如需完整联调，需要修改前端代码。

**示例：将首页轮播图改为从后端获取**

修改 `pages/index/index.js`：

```javascript
onLoad: function() {
  // 获取轮播图列表
  wx.request({
    url: 'http://localhost:8081/api/banner/list',
    success: res => {
      if (res.data.code === 200) {
        this.setData({
          bannerList: res.data.data.map(item => item.imageUrl)
        });
      }
    }
  });

  // 获取热门文章
  wx.request({
    url: 'http://localhost:8081/api/article/hot?limit=4',
    success: res => {
      if (res.data.code === 200) {
        this.setData({
          hotList: res.data.data.map(item => item.coverImage)
        });
      }
    }
  });
}
```

### 4.3 联调验证点

| 验证项 | 验证方法 |
|--------|---------|
| 后端收到请求 | IDEA 控制台打印 SQL 日志 |
| 前端收到响应 | 微信开发者工具 Network 面板显示 200 |
| 数据正确渲染 | 小程序页面显示后端返回的数据 |
| 跨域正常 | 无 CORS 报错（后端已配置 `CorsConfig.java`） |

## 五、常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|---------|---------|---------|
| IDEA 启动报错，数据库连接失败 | MySQL 未启动或密码错误 | 检查 MySQL 服务状态和 `application.yml` 配置 |
| 访问 API 返回 404 | URL 路径错误或 Controller 未扫描到 | 检查路径前缀 `/api/xxx`，确认包路径正确 |
| 微信开发者工具请求失败 | 未开启"不校验合法域名" | 详情 → 本地设置 → 勾选不校验 |
| 小程序真机调试无法访问后端 | 真机无法访问 localhost | 使用局域网 IP（如 `http://192.168.x.x:8081`） |
| 图片资源加载失败 | 图片路径错误或资源不存在 | 检查 `pages/images/` 目录下图片是否存在 |
| DBeaver 执行 SQL 报错 `FULLTEXT indexes` | MySQL 版本低于 5.6.4 | 将 `FULLTEXT KEY` 改为普通 `KEY` |
| DBeaver 连接 MySQL 失败 | 驱动未安装或配置错误 | 在 DBeaver 中下载对应版本的 MySQL 驱动 |
